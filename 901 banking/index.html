How to Open An Account/ChangePIN/Registration With *901*0#

Dial *901*0# and follow the prompts, you can either open an account by using your BVN or by providing your personal details as prompted. Upon provision of the required details, you will receive an SMS with details of your new account number..

How to Purchase Airtime *901*Amount#

Airtime TopUp

Just dial *901*Amount# (e.g: *901*1000#) on your phone, wait for the success message and start making calls. To buy airtime for friends or family members, simply dial *901*Amount*Phone Number#.

Airtime TopUp for Friends & Family

To buy airtime for friends or family members, simply dial *901*Amount*Phone Number#.

Benefit
  • Airtime can be purchased anytime, anywhere.
  • It is very easy to use.
Feature

How to Transfer Money With *901#

Transfer to an Access Bank account

Simply dial *901*AMOUNT*NUBAN Account Number# (e.g: *901*1000*1234512345#) from your phone number. Authenticate using a four (4) digit security code. You can create a security code using the Menu option.

Transfer to Other Banks

Dial *901*AMOUNT*NUBAN Account Number# (e.g *901*1000*1234512345#) from your phone. Authenticate using a four (4) digit security code. You can create a security code using the Menu option.

Benefit
  • It's fast, convenient, and secure.
  • Available 24/7.
  • It saves you the stress of queuing up in the banking halls.

How to Check Your Account Balance With *901*00#

Simply Dial *901*00# on your Access Bank registered phone number. Authenticate using a four (4) digit security code (You can create a security code using the Menu option). Your account balance is immediately generated after this has been provided.

Benefit
  • Check the status of your account any time, wherever you are.

How to Deactivate your account

Dial *901*911# on any mobile phone. Input the registered phone number on the account that you want to blacklist ...your USSD profile will be deactivated and blacklisted automatically.
